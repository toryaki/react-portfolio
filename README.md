Tory Deutsch - React Portfolio

This is a personal portfolio website built using React.js.

Project Features:
- Pages: Includes Home, About, Projects, and Contact pages.
- Routing: Uses React Router for navigation between pages.
- Styling: Styled using Tailwind CSS to be fully responsive on desktop and mobile.
- Interactivity: Includes a contact form with basic validation and simple page animations.

How to Run Locally:
1. Install the required dependencies by running: npm install
2. Start the development server by running: npm run dev
3. Open your browser and go to http://localhost:5173 to view the website.

Deployment:
This project is configured to be deployed using Vercel.
